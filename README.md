# Power Zero (Zhero) 🛡️
### The Financial Sentinel • Open Source • ISO-Standard Architecture

Power Zero is a privacy-first subscription manager designed to plug wealth leaks and reach a "Zero Waste" financial state. 

## The Mission
Most subscription trackers are built to sell your data or upsell you to "Premium." Power Zero is built to **utilize**. It is a local-first, zero-image, high-performance utility that treats your subscriptions as a "systemic leak" to be managed.

## Technical Architecture (ISO-Grade)
To ensure long-term maintainability and developer pride, this project follows a strict **Clean Architecture**:

- **Core (Domain):** Pure TypeScript. No framework dependencies. ISO 4217 (Currency) & ISO 8601 (Time) compliant.
- **Infrastructure:** Offline-first via **IndexedDB (Dexie.js)**. Data never leaves your device.
- **Presentation:** Atomic React components using **Lucide Icons**. 100% image-free for maximum performance.
- **Notifications:** Decoupled provider system (Local Web Push / ntfy / FCM).

## Features
- **Non-Judgmental Framing:** Category-based tracking (Entertainment, Learning, Tools) instead of brand-blaming.
- **The Burn Rate:** Real-time calculation of your Daily, Monthly, and Annual "Wealth Leak."
- **Sentinel Alerts:** Actionable notifications to "Keep or Kill" before a trial ends.
